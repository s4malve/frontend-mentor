## Run Locally

Clone the project

```bash
  git clone https://github.com/s4malve/frontend-mentor
```

Go to the project directory

```bash
  cd frontend-mentor
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

The content will serve on `http://localhost:3000/`
